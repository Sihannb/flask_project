# flask_project
Reinforcement Learning project

Ce projet est une pratique sur le renforcement par apprentissage (reinforcement learning en anglais). Pour le réaliser, nous avons disposé des technologies modernes : 
    * Frontend : HTML + CSS + Tailwindcss
    * Backend: Flask + SQLalchemy + Jinja2
    * Database : PostgreSQL hébergé sur Neon

    Ce projet consiste à recommander des voitures selon les likes ou vues obtenues. On propose aux clients des marques de voiture  qui existent.
